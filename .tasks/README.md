TASK_MANAGEMENT_SYSTEM
====================
This directory contains task management files for the mylm project.

Files:
- backlog.md - Main task backlog
- current.md - Current sprint tasks
- completed.md - Completed tasks archive
- workflow.md - Development workflow guidelines

Usage:
1. Add new tasks to backlog.md
2. Move tasks to current.md for active development
3. Archive completed tasks to completed.md
4. Follow workflow guidelines in workflow.md
